## Instalasi

1. Clone repository:
   ```bash
   git clone https://github.com/Arcaz22/nongki_test.git
2. Copy the .env configuration file and fill it in:
   ```bash
   cp .env.example .env
   ```
3. Run app(docker):
   ```bash
   docker-compose up --build -d
   ```
4. Thank You
